## What is NomadReady PH

NomadReady PH is an AI-powered LGU dashboard that helps a tourism office evaluate whether a destination is ready for digital nomads.

Pilot destination: **Carles, Iloilo**

## Core demo question

Is Carles ready not only for short-term tourists, but also for digital nomads who want to stay, work, and explore?

## Winning demo flow

```
Local tourism data → Readiness score → Gaps → Map → AI action plan
```

## Main user

Municipal Tourism Officer / LGU Tourism Office
